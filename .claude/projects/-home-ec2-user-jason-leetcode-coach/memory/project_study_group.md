學生參加 LeetCode study group，成員輪流分享題目。

**Why:** 需要準備到能「教別人」的程度，不只是自己會解。
**How to apply:** 當學生說「換我分享」某題時，用 jump-to 模式教學，並特別加強 How to Say It 和 Feynman Gate 的 Transfer 環節，確保能清楚向同儕解釋。
